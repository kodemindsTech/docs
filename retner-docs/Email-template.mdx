# How to Create an Email Template

## Why Email Templates?

Email templates in Retner allow you to create reusable, branded email designs for campaigns and automated journeys. With a well-designed template, you can maintain consistent branding across all your customer communications without redesigning every email from scratch.

Retner offers three editor options to suit different skill levels — from a beginner-friendly drag-and-drop builder to full HTML customization.

## Step-by-Step Guide

### Step 1: Navigate to Email Templates

Go to **Templates → Email Template** in the left sidebar. You will see:
- **Your Emails** — Saved templates you have created
- **Pre-Built Templates** — Ready-to-use designs (Abandoned Cart Reminder, Monthly Report, Course Enrollment, Thank You Message, and more)

![Email Templates Overview](/images/email-template-list.png)

### Step 2: Choose How to Create

Click the **Create from scratch** button (dropdown) to reveal three creation options:

| Option | Best For |
|---|---|
| **Drag and Drop Editor** | Non-technical users, visual design |
| **Simple Editor** | Text-based emails with images/attachments |
| **HTML Custom Code** | Developers needing full design control |

![Email Creation Options](/images/email-create-options.png)

Alternatively, you can click **Use** on any pre-built template to start from a ready-made design.

---

## Option A: Drag and Drop Editor

### Step 3a: Open the Drag and Drop Editor

Click **Drag and Drop Editor** from the dropdown. The editor opens with:
- **Elements Panel** (left) — Title, Text, Image, Video, Button, Logo, Social, HTML, Divider, Navigation, Spacer
- **Canvas** (center) — Your email design area
- **Template Name and Subject Line** fields at the top

![Drag and Drop Email Editor](/images/email-dnd-editor.png)

### Step 4a: Set Template Name and Subject Line

- **Template Name** — Enter an internal name to identify this template
- **Subject Line** — Enter the email subject line your recipients will see in their inbox

### Step 5a: Build Your Email

Drag elements from the left panel onto the canvas:

| Element | Purpose |
|---|---|
| **Title** | Add a main heading |
| **Text** | Add body copy |
| **Image** | Insert product images or banners |
| **Button** | Add a call-to-action button |
| **Logo** | Insert your brand logo |
| **Video** | Embed a video thumbnail with link |
| **Social** | Add social media icon links |
| **Divider** | Add a horizontal line separator |
| **Spacer** | Add whitespace between elements |
| **Navigation** | Add a navigation menu |
| **HTML** | Insert custom HTML blocks |

### Step 6a: Preview and Save

Use the **desktop/mobile toggle** (top right) to preview how the email looks on different devices. Click **Save** when done.

---

## Option B: Simple Editor

The Simple Editor is for text-focused emails. You can add basic formatting, images, and file attachments without complex layout design. Ideal for plain newsletters or notification emails.

---

## Option C: HTML Custom Code

If you have a custom HTML email template, select **HTML Custom Code**. Paste or write your complete HTML/CSS code directly. This gives you pixel-perfect control over the design.

<Warning>
When using HTML custom code, always test your email across different email clients (Gmail, Outlook, Apple Mail) as rendering can vary.
</Warning>

---

## Using Pre-Built Templates

Retner provides professionally designed ready-to-use templates:

- **Abandoned Cart Reminder** — Re-engage customers who left items in their cart
- **Monthly Report** — Share performance metrics with your team
- **Course Enrollment** — Welcome new course participants
- **Thank You Message** — Post-purchase appreciation email

To use one, click **Use** to load it into the editor, or **Edit** to customize it.

## Saving and Managing Templates

- Saved templates appear under **Your Emails → Saved Templates**
- You can search templates by name using the search bar
- Sort by Newest, Oldest, or Alphabetical using the sort dropdown
